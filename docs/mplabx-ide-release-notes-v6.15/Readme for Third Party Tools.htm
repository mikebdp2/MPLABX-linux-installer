1       Third Party Tools Support

MPLAB X IDE supports development tools from many different vendors. If you encounter problems using the tools with MPLAB X IDE, see the vendor website for more tool information. Also ensure you have the latest version of the tools and associated MPLAB X IDE plug-ins.

The plug-ins can be downloaded from the plug-in portal using the MPLAB X IDE plug-in manager or from the third-party vendors. See the MPLAB X IDE documentation for how to install a plug-in.

Many third-party tools and vendors are listed on the Microchip third-party website (see Section 4). Others include:

·         FreeRTOS™ (http://www.freertos.org)
Note: The 8-bit FreeRTOS is not supported by Microchip’s RTOS viewers.

·         MATLAB/Simulink (http://www.mathworks.com/)

·         Gimpel PC-LINT/MISRA (http://www.gimpel.com/html/pcl.htm)

·         SEGGER J-Link (http://www.segger.com/jlink-debug-probes.html)

·         Proteus VSM MPLAB Viewer (http://www.labcenter.com/support/vdmmplab.cfm)

·         LDRA Rules for MPLAB X IDE(http://ldra.com/industrial-energy/products/ldrarules/ldrarules/)

2       Version Control Systems Support

Source control program support in MPLAB X IDE includes:

·         GIT – https://git-scm.com/

·         Subversion – https://subversion.apache.org/

·         Mercurial – https://www.mercurial-scm.org/

·         CVS* – http://savannah.nongnu.org/projects/cvs

* For CVS, see under Tools>Plugins, Available Plugins tab.

Through the use of external plug-ins, support for other VC programs may be added.

7       Known Problems

XPI-53: [SEGGER J-Link] Breakpoints set in the data memory using SFRs with or without pass count do not halt the target.

XPI-49: [SEGGER J-Link] The MPLAB X IDE dashboard does not reflect the number of set breakpoints correctly when using SEGGER J-Link probe. Even when a breakpoint is used, all breakpoints are shown as available.

XPI-47: [SEGGER J-Link] Error messages reported using MPLAB X IDE and SEGGER plugin are cryptic.
