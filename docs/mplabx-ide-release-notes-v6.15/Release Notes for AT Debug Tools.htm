2       Atmel-ICE

Atmel-ICE (ATATMEL-ICE) is a powerful development tool for debugging and programming ARM® Cortex®-M based SAM microcontrollers and AVR® microcontrollers with On-Chip Debug (OCD) capability.

For more information on this tool, see:

  • Product Webpage: https://www.microchip.com/developmenttools/ProductDetails/ATATMEL-ICE
  • Developer Help: https://microchipdeveloper.com/atmelice:start
  • In the IDE, Help>Help Contents>Atmel-ICE In-Circuit Debugger.

3       Power Debugger

Power Debugger (atpowerdebugger) is a powerful development tool for debugging and programming AVR microcontrollers using UPDI, JTAG, PDI, debugWIRE, aWire, TPI or SPI target interfaces and ARM® Cortex®-M based SAM microcontrollers using JTAG or SWD target interfaces.

For more information on this tool, see:

  • Product Webpage: https://www.microchip.com/developmenttools/ProductDetails/atpowerdebugger
  • The Power Debugger WebHelp

4       Embedded Debugger

The Embedded Debugger (EDBG) is an onboard debugger for integration into development kits with AVR or SAM MCUs. EDBG enables debugging of a target device without an external debugger. The EDBG is embedded on all Xplained Pro evaluation kits, although not all functionality is available on all kits.

For more information on this tool, see:

  • Xplained kits: https://www.microchip.com/xplained
  • The EDBG User’s Guide (PDF).

5       SAM-ICE

End of life (EOL) product and can no longer be ordered. For MPU and SAM developers, SAM-ICE has been replaced by the J-32 Debug Probe (DV164232). Alternatively, SAM developers can use either Atmel ICE (ATATMEL-ICE) or MPLAB ICD 4 (DV164045).

SAM-ICE is a JTAG emulator designed for SAMA5, SAM3, SAM4, SAM7 and SAM9 ARM® core-based MCUs and MPUs, including Thumb mode.

For more information on this tool, see:

  • Product Webpage: https://www.microchip.com/developmenttools/ProductDetails/AT91SAM-ICE
  • OEM Information: https://www.segger.com/products/debug-probes/j-link/models/other-j-links/j-link-oem-versions/
